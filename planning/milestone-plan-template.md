# Milestone Plan Template

Use this template after completing `intake/problem-intake.md`.

The default plan runs from `v0.1.0` to `v1.0.0`. The user may change the number, names, or scope of milestones before the agent creates GitHub milestones and issues.

## Planning Rules

- The agent drafts the plan.
- The user reviews and edits the plan.
- The agent creates GitHub milestones and issues only after explicit user approval.
- Each milestone should produce something testable with synthetic data.
- GitHub Actions should be minimal and based on actual project requirements.
