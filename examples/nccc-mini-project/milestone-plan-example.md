# Sample Milestone Plan

## v0.1.0 - Import and Review Synthetic Inventory

- Import a small synthetic CSV file inventory.
- Display or print a reviewable list of items.
- Let the user mark `pending`, `reviewed`, or `follow_up`.
- Test the flow with the included synthetic CSV file.

## v0.2.0 - Track Synthetic Hash Status

- Add fields for `hash_recorded` and `hash_algorithm`.
- Show missing-hash items in a separate filtered view or report.
- Verify the workflow with expanded synthetic examples.

## v0.3.0 - CASE/UCO/CAC Mapping Review

- Document which CASE/UCO/CAC concepts fit the current data model.
- Add a lightweight export or mapping draft for future interoperability.
- Review naming and relationships before adding more features.
