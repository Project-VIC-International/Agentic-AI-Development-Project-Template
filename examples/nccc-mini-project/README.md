# NCCC Mini Project Example

This example shows a deliberately small, safe project that matches the course workflow without requiring real investigative data.

## Example Idea

Build a simple file inventory tracker that:

- Accepts a synthetic list of files and folders.
- Tracks a review status for each item.
- Records a synthetic hash-computed flag.
- Exports a small CASE/UCO/CAC-aligned summary later if needed.

## Included Files

- `problem-intake-example.md` shows how a student might scope the problem.
- `milestone-plan-example.md` shows a small reviewed roadmap.
- `synthetic-data/sample_file_inventory.csv` is safe sample input data.

Do not copy the example blindly. Use it as a pattern for creating your own intake, milestone plan, and synthetic dataset.
