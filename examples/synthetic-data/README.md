# Synthetic Data Examples

This folder is for harmless, fictional examples that help test project workflows.

The included `sample-case-uco-cac.jsonc` file is a commented teaching example that shows a small CASE/UCO/CAC-style JSON-LD graph. It is not strict JSON because the comments explain what each part does.

Before using the example with a strict JSON parser, JSON-LD processor, or CASE/UCO validator, remove the comments or ask the agent to generate a clean `.json` version. For real projects, ask the agent to validate and refine class/property choices with CASE/UCO SDK v1.11.0+ and its MCP server.

Replace the example with synthetic data that matches your project, while keeping all real case data and sensitive operational details out of the repository.
