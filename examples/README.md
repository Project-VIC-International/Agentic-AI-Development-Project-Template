# Examples

This folder contains small examples that help users and agents understand the template.

## Files

- `sample-milestone-plan.md` shows the shape of a simple milestone plan.
- `synthetic-data/` contains safe synthetic data examples.
- `synthetic-data/sample-case-uco-cac.jsonc` is a commented CASE/UCO/CAC-style JSON-LD teaching example.

## How to Use These Examples

Use the examples as references, not as real mission data. Ask the agent to adapt them to your `intake/problem-intake.md`, then validate ontology choices with CASE/UCO SDK v1.11.0+ and its MCP server.

See also:

- `docs/synthetic-test-data.md`
- `docs/case-uco-cac-guidance.md`
- `planning/milestone-plan-template.md`
