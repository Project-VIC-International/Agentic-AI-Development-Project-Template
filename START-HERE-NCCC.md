# Start Here for NCCC Students

This repository is your working project starter.

Before asking Cursor to build anything:

1. Read `QUICKSTART.md`.
2. Read `SECURITY.md`.
3. Complete `intake/problem-intake.md`.
4. Ask Cursor to summarize `.cursor/rules/`.
5. Ask Cursor to review your intake form.
6. Ask Cursor to create a milestone plan.
7. Only then begin implementation.

Do not paste real case data into Cursor, GitHub, issues, logs, screenshots, test files, or sample data. For this course, all development must use synthetic data.
